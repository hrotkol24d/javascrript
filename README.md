### 1.1.3 JavaScript as an a interpreted language
interpreted (parancsértelmezős) languages:
- JavaScript (Böngésző, böngészőbe épített JS engine)
- Python (python.exe)
- PHP (php.exe)

compiled (forditós) languages:
- Java
- C#
- C/C++
